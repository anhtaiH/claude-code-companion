# Roadmap

## V1

- Read-only review and adversarial review.
- Read-only diagnosis and planning tasks.
- Local job board.
- Claude session ID handoff.
- MCP tools for Codex.
- MCP prompt templates for slash-command and command-palette style hosts.
- Primary `consult` tool for agent-native delegation.
- Public docs and tests.

## V1.x

- Better Codex plugin installer story when local plugin distribution stabilizes.
- Optional repo-local policy file for review defaults.
- More structured review output rendering.
- More background-job observability.
- Configurable context limits.

## V2

- Optional read-only Claude tool profile using `Read`, `Glob`, `Grep`, and
  tightly scoped git commands.
- Live agent-to-agent handoff flow.
- Better multi-agent conversation transcript model.
- Optional write-capable rescue mode only if the user explicitly enables it and
  the security model is redesigned around that risk.
